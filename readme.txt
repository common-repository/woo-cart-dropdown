Adds WooCommerce cart and dropdown to any menu.

== Description ==

This plugin adds a cart button to all menus selected in the settings page. The button can be configured to display any combination of information and includes dropdown cart contents. 

The plugin includes the following options:

* Display in multiple menus.
* Toggle display individually for [ Icon, Counter, Text, Seperator, Amount ].
* Position the cart first or last in the menu order, or float it left or right.
* Choose between 10 cart icons.
* Toggle displaying dropdown cart contents.
* Choose to display cart only when it has items in it.

Compatibility:

* WooCommerce

== Installation ==

Delete any old installations of the plugin. Extract the zip file and drop the contents in the wp-content/plugins/ directory of your WordPress installation. Activate the plugin from Plugins page.

Once the plugin is activated navigate to WooCommerce > Cart Dropdown. Select the menus you want the cart to display in, save and you are all set.
